How arrays work

I like to think of an array as a shelf full of items. The items can be strings, integers, floats, or even more arrays (shelves). An array can be defined like so:

array = ["i'm", "just", "dying", "in", "your", "arms", "2" "-nite."]

Arrays can be manipulated through various predefined methods as well as iteration techniques. Say I wanted to bunch up my array into one line of glorious 80's nostalgia. I could define a method to do that:

def nostalgia(array)
array.join " "
puts array + " Must have been something you said."

But that's just one method. The list of built in string manipulation methods extends for miles and the mastery of methods is something that takes great time and research. For that reason, I won't go terribly in depth.

If I wanted to call up just one piece of my array, I would do that by grabbing the index. The index system looks something like this:

array = ["Jen", "Ted", "Pat", "Roy"]
index #    0      1      2      3

The first item in the array has the hash of 0 and it goes up from there. Say I wanted to call Pat's string, I would type:

array[2]

...and that would return "Pat". I could also call the last item in a string by typing:

array[-1]

...and that would return "Roy".



How hashes work

Hashes are like bigger and more organized cousin of the array. A hash consists of a list of key/value pairs which

Say I have a list of silly names and the phone numbers which are paired to each name. The hash could be defined like so:

phone_directory = {
  "Bob Bobbins" => 9152940564
  "Glob Globbins" => 9159859240
  "Sheral Muxlix" => 8492958483
  "Hob Squazin" => 4149285483
}

In this example, the names are the 'keys' which are paired with the phone numbers which are the 'values'. You can call a value from the previous hash using this value.

hash_name["key_name"]

This would return the matching value of the key. So...

phonedirectory["Bob Bobbins"]

Would return Bob's phone number, 9152940564.

There are also ways to iterate across keys and values of hashes. I could organize the names alphabetically with the following method:

Hash[phone_directory.sort]

Just as with arrays, hashes too have hundreds of different predefined methods for iterating and organizing so once again, I won't delve too in depth.

These are just a few of the main elements in ruby. Between strings and arrays and hashes, you should have your hands full for quite some time. I know mine are full right now.
