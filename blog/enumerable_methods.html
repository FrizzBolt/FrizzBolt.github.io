"Cycle", "Method", "Cycle", "Method"

The #cycle method is a handy method for iterating through collection and applying a block of code repeatedly on said colelction. The #cycle method has a fairly simple syntax. The method is run on the collection and an optional argument is taken in. On the same line, the method runs a block on each separate item in the collection and re-iterates the block over the amount of times listed in the argument. If there is no argument defined, the cycle will repeat indefinitely. If any number other than a positive integer is entered as an argument or the collection is empty, nothing is returned.

Like most things in programming, it's easier to explain with an example:

array = ["cycle", "method"]
array.cycle(2) { |x| print x }
=> will return "cycle", "method", "cycle", "method"

That's lovely, but I'm thinking maybe you'd rather crash Ruby. In that case, you'd want to remove the argument:

array = ["cycle", "method"]
array.cycle { |x| print x }
=> will return an infinite loop of "cycle", "method", "cycle", "method"

Yay! We did it! Now close out Ruby and do it all over again!
