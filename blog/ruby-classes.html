Classy Stuff

class Superhero

     def initialize(name, gender, secret_identity, arch_nemesis, superpower)
          @name = name
          @gender = gender
          @secret_identity = secret_identity
          @arch_nemesis = arch_nemesis
          @superpower = superpower
     end

     def reveal_identity
          puts "A little bird told me that #{@name} is really #{@secret_identity}"
          puts "I sure hope you won't do anything unsavory with this information, #{@arch_nemesis}"
     end

     def headline_generate
          if @gender = female
               nomenclature = "her"
          else
               nomenclature = "his"
          end
          puts "#{@name} defeats #{@arch_nemesis} once again by using #{nomenclature} #{@superpower}!"
     end

end

Ok. So this is a class. What, you ask, is a class? Good question. Every object follows a hierararchy that places it under the classification of another object. The object that is a parent of an object is, in the case of coding, referred to as a class. A class can contain one or more methods as well as other classes - a class containing classes is called a superclass.

Since classes contain more than one method, they also usually contain separate forms of variables called instance and class variables. See how these variables have an @ symbol in front of them?The @ symbol before the variable defines an instance variable. An instance variable is unique to the instance of the class. A class variable, on the other hand would be defined with two consecutive @ symbols (@@). Class symbols are symbols that are unique to the class itself and to not get assigned to individual instances. The #initialize method is a method thats unique to classes which assigns instance variables to incoming instances of the class.

So, say I create a new instance of the class Superhero, I would include the the initialization arguments as my arguments in my new instance. So say I wanted to create a new superhero called "Organs on the Outside Man" - I know, gross. I couldn't think of a better one, but I'm a techie, not a comic-book artist - I would create it with this line:

organs_outside = Superhero.new("Organs on the Outside Man", "male", "Clive Gutman", "Pants-For-Hands Girl", "liver failure")

I know that's a verbose example and a weird one at that, but I wanted to have some fun with this. Now, if I call the method headline_generate on the organs_outside variable, I'll get something like this:

organs_outside.headline_generate
returns=> "Organs on the Outside Man defeats Pants-For-Hands Girl once again by using his liver failure."

Alright, that's a really disjointed and weird headline and I'm not quite sure how you can defeat someone by making your own liver fail, but hopefully you get the idea. Or maybe you don't. E-mail me if you don't. I need some sleep.
